# 楽天イーグルス戦力分析

## 分析の目的
「楽天イーグルスが上位に食い込むには何が必要か」をデータで明らかにする

## 使用ツール
- SQLite / DBeaver（データ加工・分析）
- Tableau Public（可視化）※後日追加予定

## 分析結果サマリー

### 課題① 打線の長打力不足
- チームOPS・本塁打がリーグ最下位
- 主砲・浅村栄斗のOPSが0.662と低迷
- 外国人野手ボイトが機能不全

### 課題② 先発は良いのに勝てない
- 早川・瀧中は防御率2点台と優秀
- 古謝・荘司はQSを稼ぐのに勝ちがつかない
- 打線の援護不足が直接の原因

## 優勝するための提言
- 長打力のある外国人野手の獲得
- 浅村の復調
- 中継ぎ陣の整備
